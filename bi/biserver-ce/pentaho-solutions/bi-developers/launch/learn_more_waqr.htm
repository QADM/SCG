Ad Hoc Reporting

The Pentaho ad hoc reporting interface is an interactive, AJAX-based, Web client that delivers self-service reporting to business users of any skill level. Powered by Pentaho metadata, the ad hoc reporting interface provides:

  • Simple, four-step wizard for creating template based reports
  • Report generation in a variety of formats including PDF, HTML, Excel, and RTF
  • Metadata-based query creation without requiring end users to know SQL
  • Interoperability with Pentaho Report Designer file format

Centralized Metadata Layer

To use the ad hoc reporting interface with your own data, you must design a metadata model using the Pentaho Metadata Editor. Pentaho metadata provides the following features:

  • For designers, the ability to create one report that satisfies multiple end-user needs
  • For business users, the ability to create their own ad hoc reports based on centralized business rules, without any technical skills or knowledge of SQL
  • Centralized maintenance for data connectivity, security, formatting, and more.
 

To launch the Pentaho Metadata Editor and begin designing your own models:

Windows

From the Windows Start menu, select Start > Programs > Pentaho > Pentaho Reporting > Metadata Editor

 

Linux

Run metaeditor.sh found in the metadata-editor directory under your root Pentaho installation directory
