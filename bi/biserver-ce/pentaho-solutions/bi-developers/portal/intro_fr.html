Bienvenue dans la démo du portail Pentaho BI

La démonstration du portail Pentaho BI utilise JBoss Portal pour montrer comment les composants Pentaho peuvent être utilisés dans des pages portail.

La plateforme Pentaho BI inclut ces portlets d'exemple, compatibles JSR-168, pour montrer comment nos composants BI peuvent être encapsulés dans les pages web d'une autre application.

Utiliser les options du menu ci-dessus pour tester les exemples que nous avons fourni.

Lisez le document 'Creating Pentaho Solutions' pour obtenir des informations sur la maniére de modifier les exemples ou de construire votre propre systéme.
